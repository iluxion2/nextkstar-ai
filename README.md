# AI Face Revolution - Sociotechnical Mirror

A revolutionary AI-powered facial analysis platform that explores beauty bias, digital identity, and media influence. This project combines cutting-edge machine learning with social science research to provide insights into how AI perceives beauty across different cultural standards.

## 🌟 Features

### Core Analysis
- **AI Beauty Score**: Advanced facial analysis using 68-point landmark detection
- **Facial Features Breakdown**: Symmetry, skin clarity, proportions, and expression analysis
- **Personality Traits**: Confidence, friendliness, intelligence, and attractiveness assessment

### Revolutionary Features
- **Bias Analysis Mode**: Discover how AI perceives your face across different cultural standards
- **Celebrity Lookalike Matching**: Find your doppelgängers from K-pop idols to anime characters
- **Digital Doppelgänger**: Generate AI-morphed versions of yourself as different cultural archetypes
- **Cultural Lens**: Explore beauty standards across Korean, Western, Global, and Anime aesthetics

### Social & Research
- **Global Leaderboard**: Compare your results with users worldwide
- **Research Tool**: Contribute to AI bias studies and cultural analysis
- **Privacy First**: Images are processed securely and deleted unless shared publicly

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/ai-face-revolution.git
cd ai-face-revolution
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
```

3. **Run the development server**
```bash
npm run dev
# or
yarn dev
```

4. **Open your browser**
Navigate to [http://localhost:3000](http://localhost:3000)
